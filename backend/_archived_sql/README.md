# Archived SQL Schemas

These SQL files were from early PostgreSQL planning. The platform uses **MongoDB** exclusively.

They are archived here for reference only. **Do NOT use these as source of truth.**

The actual schema contracts are defined in:
- `backend/src/schemas/collections.js` — field definitions, types, indexes
- `backend/src/schemas/validate.js` — validation utility
- `docs/mongodb-schema.md` — human-readable documentation
